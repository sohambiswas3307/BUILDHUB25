🔍 HOW TO VIEW COMPLETE DATABASE DETAILS
═══════════════════════════════════════════════════════════════════════════════════════════════

Method 1: Using DB Browser for SQLite (GUI - Easiest)
───────────────────────────────────────────────────────────────────────────────────────────────
1. Download DB Browser for SQLite from: https://sqlitebrowser.org/
2. Install the application
3. Open DB Browser for SQLite
4. Click "Open Database"
5. Navigate to: D:\app labour\buildhub.db
6. Click on "Browse Data" tab
7. Select "users" table
8. View all registered users with complete details

Method 2: View Through Application
───────────────────────────────────────────────────────────────────────────────────────────────
1. Run: mvn javafx:run
2. Click "Sign Up" to create new users
3. Use the registered email to login

Method 3: Register New Users to See Full Details
───────────────────────────────────────────────────────────────────────────────────────────────
The application stores:
- Name
- Email (Login credential)
- Password (BCrypt encrypted)
- Role (customer, contractor, labour)
- Phone
- Address
- Additional fields based on role
